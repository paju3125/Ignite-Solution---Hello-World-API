## Installation


1. Create a virtual environment
```bash
  python -m venv venv
```

2. Activate environment
```bash
  .\venv\Scripts\activate
```

3. Install dependencies
```bash
  pip install -r requirements.txt
```

4. Run flask webapp
```bash
  python app.py
```

5. Interact with webpage
```bash
  Click on - http://localhost:5000
```
